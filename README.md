# mountainScannerJS

- uses nodeJS for the `requestAnimationFrame` function
- [midijs library](https://webmidijs.org/docs/) works on Google Chrome only!

## run

- `npm start` to just start
- `yarn server` to start & keep track of code changes (dev mode)
